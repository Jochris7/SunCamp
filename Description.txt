Projet 2 : Application web pour colonie de vacances

•	Présentation des colonies, voyages, activités.

•	Formulaire de contact, réservation en ligne.

•	Paiement en ligne ou en agence.

Partie admin : gérer les voyages, réservations, paiements, messages reçus.



--------------------------------------------------------------------------

Presentation de colonies, voyages, activités = Photo + titre + texte de description + prix 

Formulaire de contact = Niveau front avec different champs tels que : nom complet + adresse mail + suejt ( Inscription des camps ..herbergement etc ) + le description et explication maintenant dans (Qu'est ce que nous pouvons faire pour vous )

Réservation en ligne = Choix du camps + participants (enfants adultes etc et le nombres de personnes concerné) apres avoir choisir ces parametre vous remplirez les formulaires (prénoms , noms , date de naissance , allergie/infos medicale)

Paiement en ligne = carte bancaire ou mobiel money l'nterface selon le choix 

----------------------------------------------------------------------------

Donc voici un peu les model je propose : 
- Admin : email mot de passe 
- Colonie : titre , images , description , prix , durée de la colonie , destinantions (voyages ) , actvitéé
- Contact : nom complet du contacteur ...
